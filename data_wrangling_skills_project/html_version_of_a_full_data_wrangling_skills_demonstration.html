Project: Wrangling and Analyze Data¶

Introduction¶

In this project, we aim to collect data from 3 sources, which is in different formats. We aim to showcase data wrangling skills in gathering the data, assess its quality and tidiness, then clean it. We will then store it and use it for analysis and visualization. On the visualization we will create insightful and trustworthy analysis on the wrangled data.

The datasets used in this project¶

The 3 datasets used in the project are:

  1. Enhanced Twitter Archive The WeRateDogs Twitter archive contains basic tweet data for all 5000+ of their tweets. One column the archive does contain each tweet's text, which was used to extract rating, dog name, and dog "stage" (i.e. doggo, floofer,pupper, and puppo)

  2. Additional Data via the Twitter API Retweet count and favorite count are two of the notable column omissions whose data can be gathered from Twitter's API. We are going to query Twitter's API to gather this valuable data.

  3. Image Predictions File It is a table full of image predictions (the top three only) alongside each tweet ID, image URL, and the image number that corresponded to the most confident prediction (numbered 1 to 4 since tweets can have up to four images).
